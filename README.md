# 优住网门店版前端

用户网门店版本项目代码仓库，该项目主要用于不用门店使用，区别于优住网平台版本，酒店客服可以配置自己的公众号，通过公众号菜单跳转到该项目。

# 项目技术栈
描述            | 技术
---------------|-------
数据展示界面框架 | React
页面路由        | React-Router(v3)
全局状态管理    |  Redux
代码打包        | webpack
样式代码        | sass(scss)
支付封装        | ping++
网络请求        | fetch
